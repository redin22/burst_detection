# Burst Detection
Burst detection using CUSUM change-point detection and DBSCAN clustering

This project involves the implementation of 2 burst detection methods: CUSUM change-point detection on pressure time-series for areas with sensor coverage, and customer contact spatial/temporal clustering for areas without sensors. The final outputs are 2 alert feeds generated by each of the detection methods to show potential incidents, their severity, advisory radius, and the number of pipes affected. You will also get 2 validaion results reports showing false positives, false negatives, precision and recall for each of these two methods against the provided validation set.
